## Prerequisite
1. PHP 7.4
2. MYSQL 5+
3. Composer 1.9.2

## Install dependecies
1. composer install

## Steps to follow to run the test cases for Supermarket Checkout
1. [Import database] 
- SQL file is in the root directory, filename:- supermarket.sql, Database name must be "supermarket".
- Database credentials are mapped in .env file

2. [Command to run test cases]
- To run test cases, please execute the command:- php artisan test
- Test cases are written in "SupermarketCheckout\tests\Feature\SupermarketCheckoutTest.php"

3. [Debugging Code]
- For debugging individual scenarios one command is created in "SupermarketCheckout\app\Console\Commands\ScanProductsToCheckout.php"
- Command:- php artisan command:name
